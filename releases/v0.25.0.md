# v0.25.0 — 2026-06-12

## Added

- Mechanics: `.claude/workflows/wave-pipelines.js` — `/wave` now executes its pipelines through a saved Workflow script: plan/arch/dev parallel across a group, gitter SETUP, QA gate, and merge tail serialized via one lock, `dependsOn` skip semantics, journal resume. The executable payload persists to `$WAVES/{wave-name}/workflow.json` (wave.md Step 0d template) and Step 1 executes from that file verbatim. The sequential `Skill("build")` path remains as a transitional fallback. (safe-auto)
- Mechanics: `/animate` — new command: research-grounded educational HTML animation of any structure or flow. Five-step procedure: cited fact sheet (a fact without a citation does not exist) → storyboard with fact-coverage table → spawned builder → dual blocking verification (Playwright behavior pass + fresh-agent accuracy diff of every rendered string against the fact sheet) → delivery. Ships with `docs-commands/animate/references/engine.md` — the debugged engine contract (pure function-of-time render, ⚓ invariants: idempotent render, act-jump resumes playback, study mode, reduced-motion), UI chrome, headless recording recipe (playwright-core + ffmpeg GIF palette pipeline), and pitfalls. (safe-auto)
- Scripts: `templates/host-swap/` — per-chat multi-account billing swap, host-level tooling (lives in the user dir, not repo `.claude/`): `/swap` slash command with account picker, per-session Keychain credential rewrite, statusline badge. Account table is parameterized per install — the blueprint ships the mechanism. (opt-in)
